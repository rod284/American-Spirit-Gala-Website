## 🚀 How to Keep Counters Updated

### Ticket Counter (in website)
Located in Sponsorship section:
- Shows "127 of 160" 
- Shows "33 remaining"
- Shows "79.4% capacity"
- Updates team AND website visitors

**To update:**
1. Edit line in `index.html`: `<strong style="color: #ffed4e;">127 of 160 tickets sold</strong>`
2. Change "127" to current number
3. Update "33" to: 160 - new number
4. Update "79.4%" to: (new number ÷ 160) × 100
5. Save file (auto-deploys to Netlify)

### Revenue Counter
Located in Sponsorship section:
- Shows "$31.7K of $45K"

**To update:**
