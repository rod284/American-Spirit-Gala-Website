===============================================================
📊 WHERE TO GET YOUR NUMBERS
===============================================================

TICKETS SOLD
→ Source: Attendee_Tracking.csv or Donorbox dashboard
→ Field: Count of registered attendees
→ Example: 127

SPONSORSHIPS COMMITTED
→ Source: Sponsorship tracking
→ Field: How many sponsorship tiers sold (not dollars)
→ Example: 13 (Captain's Helm, Band Sponsor, etc.)

SPONSORSHIP REVENUE
→ Source: Sponsorship spreadsheet
→ Field: Total dollars from all sponsors
→ Example: $25,375 (all tiers added together)

TICKET REVENUE
→ Math: Tickets Sold × $50
→ Example: 127 × $50 = $6,350

AUCTION/RAFFLE REVENUE
→ Source: Auction items sold, raffle tickets sold
→ Example: Live auction: $3,200, Silent: $1,500

===============================================================
🔒 DATA STORAGE & BACKUP
===============================================================

WHERE IS DATA STORED?
→ Browser's local storage
→ Safe & secure
→ Data persists even after closing browser

BACKUP PROCEDURE (Monthly):
1. Open Admin Panel
2. Screenshot all fields
3. Save screenshot to: Backups folder
4. OR manually enter numbers into Excel

RESTORE FROM BACKUP:
1. Open Admin Panel
2. Type in backup numbers
3. Click Save
4. Data restored!

IMPORTANT:
⚠️  Use SAME computer for updates
⚠️  Each device stores its own copy
⚠️  If using different computer, re-enter numbers
✓ Monthly backups recommended

===============================================================
❓ TROUBLESHOOTING
===============================================================

PROBLEM: Can't find Admin Panel
SOLUTION:
→ Scroll to very bottom of website
→ Look for blue "Admin Panel" link
→ Or type in URL: https://allhandsondecknhn.netlify.app/#admin-panel

PROBLEM: Changes aren't showing
SOLUTION:
→ Make sure you clicked "Save & Update Live Site"
→ Refresh website (Ctrl+R)
→ Check you saved on same computer you're viewing from

PROBLEM: Panel won't open
SOLUTION:
→ Clear browser cache (Ctrl+Shift+Del)
→ Try different browser
→ Check JavaScript is enabled (usually is)

PROBLEM: Lost data / Changed computers
SOLUTION:
→ Data is per-device
→ Re-enter numbers on new computer
→ Keep backup spreadsheet as master copy
